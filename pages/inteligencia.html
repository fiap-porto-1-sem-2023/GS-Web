INTELIGÊNCIA ARTIFICIAL GENERATIVA

O QUE É IA GENERATIVA?

Inteligência Artificial é um ramo da ciência da computação que busca criar sistemas capazes de realizar tarefas que geralmente apenas um humano seria capaz de fazer.

Existem dois tipos de IA. A discriminativa aprende a tomar decisões com base em dados e pode ser usada para classificar imagens ou prever valores. Já a IA generativa é capaz de criar coisas novas a partir das entradas.

FUNCIONAMENTO

O funcionamento mais comum de uma Inteligência Artificial Generativa envolve o uso de redes neurais. No caso das Redes Adversariais Generativas temos duas redes neurais que competem entre si.

Uma rede cria dados parecidos com os de treinamento. A outra busca distingui-los. Em cada iteração a rede geradora de dados vai ficando cada vez melhor, até o ponto de criar dados novos, porém parecidos com o conjunto de entrada.

EXEMPLOS PRATICOS DE APLICAÇÃO NA AGRICULTURA

A aplicação mais comum consiste em criar modelos de simulação de condições ambientais para depois prever o impacto desse cenário no rendimento das culturas, ajudando o agricultor na tomada de decisões.

A IA generativa também poderia ser usada para criar novas imagens reais de plantas doentes, as quais seriam usadas no treinamento de IAs de classificação e identificação de pragas.
